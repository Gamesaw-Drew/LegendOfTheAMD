Alpha 1.4.0 (Pre-Release 042817)

This is the first pre-release version for alpha 1.4.0.

Alpha 1.4.0 will be a large update, so expect a few pre-releases over the next couple of months.

Here are the currently added features. This is far from how 1.4.0 will be, it is just the beginning.

  • Updates
  • Updated engine version
  • Removed all multiplayer code as it has to be redone due to Unity updates.
  • Revamped look of main menu
  • Drastically revamped graphics (May run slowly on lower machines, this will be fixed next pre-release. At this moment, it currently keeps all the new graphics settings the same, even on lowest settings, which is a bug.)
  • First implementation of aiming and shooting - Hold right click to aim / move the camera, and use left click to shoot. You can currently kill the cogs at this point
  • Health system
  • Hitbox system
  • Early ability to drive cars - For now you will use the switch in the top corner
  • Made the map much bigger (It will have actual detail in the future)
  • Added some placeholder roads to show you the way to AMD Store
  • Added early day and night cycle
  • Added placeholder models for Chris's house / AMD Store
  • Texture Upgrades / Tweaks
  • Remade some animations

  • Known Bugs
  • Some graphic effects don't change in quality with different graphics settings
  • The car sometimes won't move forward, if it doesn't move forward, back up a tiny bit and then go forward and it should work
  • Occlusion data is out of date, some items may show or hide incorrectly
  • Nav Mesh data is out of date, cogs will only walk in a small area and may not notice some walls
  • Occasional crashes

New Launcher


  • A brand new launcher has been released, if you are using the old one, please download the new one from the website.
  • The new launcher has proper auto updating, meaning it will only download what is necessary for the update
